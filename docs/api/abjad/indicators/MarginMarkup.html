MarginMarkup¶

class abjad.indicators.MarginMarkup.MarginMarkup(*, context: str = 'Staff', format_slot: str = 'before', markup: Optional[Union[str, abjad.markups.Markup]] = None)¶

Margin markup.

>>> staff = abjad.Staff("c'4 d'4 e'4 f'4")
>>> margin_markup = abjad.MarginMarkup(
...     markup=abjad.Markup('Vc.'),
...     )
>>> abjad.attach(margin_markup, staff[0])
>>> abjad.show(staff, align_tags=89)

Set markup to externally defined command string like this:

>>> staff = abjad.Staff("c'4 d'4 e'4 f'4")
>>> margin_markup = abjad.MarginMarkup(
...     markup=r'\my_custom_instrument_name',
...     )
>>> abjad.attach(margin_markup, staff[0])
