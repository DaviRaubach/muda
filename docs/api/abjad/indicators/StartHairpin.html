StartHairpin¶

class abjad.indicators.StartHairpin.StartHairpin(shape='<', *, direction: Optional[abjad.enums.VerticalAlignment] = None, tweaks: Optional[abjad.overrides.TweakInterface] = None)¶

Hairpin indicator.

>>> staff = abjad.Staff("c'4 d' e' f'")
>>> abjad.attach(abjad.Dynamic('p'), staff[0])
>>> abjad.attach(abjad.StartHairpin('<'), staff[0])
>>> abjad.attach(abjad.Dynamic('f'), staff[-1])
>>> abjad.override(staff).DynamicLineSpanner.staff_padding = 4.5
>>> abjad.show(staff)
